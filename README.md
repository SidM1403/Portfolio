---

## 🚀 System Architecture (Tech Stack)

### **Frontend & Core**
- **Framework**: React 19 (Modern Hooks & Architecture)
- **Build Tool**: Vite (Lightning fast HMR)
- **Styling**: Tailwind CSS 4 (Next-gen utility-first CSS)
- **Rendering**: WebGL via Three.js & React Three Fiber

### **Interactivity & Animation**
- **Framer Motion**: Smooth layout transitions and scroll-triggered animations.
- **GSAP**: High-performance cinematic sequences and complex timelines.
- **WebGLBackground**: Interactive 3D particle systems and neural pulse grids.

### **Utilities**
- **Lucide React**: Clean, consistent iconography.
- **Howler**: (Optional) For futuristic UI audio feedback.
- **CVA & Tailwind Merge**: For robust, type-safe component design systems.

## ⚡ Key Features

### **1. Neural Boot Sequence**
A custom loading system that initializes the portfolio with a "CLICK TO INITIALIZE SYSTEM" prompt, setting the high-tech tone immediately.

### **2. Dynamic Data Identity**
Integrated biometric scanning animations and HUD (Heads-Up Display) overlays that frame the user experience, making it feel like a sophisticated OS.

### **3. Mission Logs (Projects)**
A dedicated section [Missions](file:///d:/ImpProject/portfolio/src/sections/Missions.jsx) showcasing various "Contracts" with:
- WellConnect: MERN Health Platform.
- Real-Time Score Tracker: Live sports data platform.
- Event Management System: Full-stack PHP/MySQL solution.
- Project Management: Role-based admin/user dashboards.

### **4. System Systems (Skills)**
An interactive skills section with:
- **Logo Marquee**: A seamless loop of mastered technologies.
- **Holographic Cards**: Detailed skill breakdowns with animated proficiency bars.
- **Interactive Circuitry**: Visual pulse effects that react to user hovering.

### **5. Temporal Sequence (Experience)**
A vertical timeline tracking academic and professional milestones with a "temporal stream" background effect.

### **6. Encrypted Transmission**
A stylized contact form with "Encryption" processing states, allowing for secure message uplinks.

## 🛠️ Setup & Initialization

To get the system running locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/SidM1403/portfolio-next-gen.git
   cd portfolio-next-gen
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start Development Environment**:
   ```bash
   npm run dev
   ```

4. **Build for Production**:
   ```bash
   npm run build
   ```

## 🖼️ Project Image Integration

To add your project images:
1.  Prepare your images (recommended size: 800x450 or 16:9 aspect ratio).
2.  Name them exactly as follows:
    - `wellconnect.png`
    - `scoretracker.png`
    - `eventmanagement.png`
3.  Create a folder named `projects` inside the `public/` directory (i.e., `public/projects/`).
4.  Place your images inside that folder.
5.  The **Missions** section will automatically detect and display them.

---

## 📄 CV Integration

To enable your CV download:
1.  Obtain your CV in PDF format.
2.  Rename the file to `cv.pdf`.
3.  Place it in the `public/` directory of this project.
4.  The "DOWNLOAD CV" button in the **About Me** section will now automatically serve this file.
